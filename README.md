# 🍔 Food Ordering System (Java)

A simple console-based **Food Ordering System** developed in **Java** to demonstrate Object-Oriented Programming (OOP) principles and clean code practices.

This project allows users to:
- View a food menu
- Select multiple food items
- Calculate the total order price
- Choose a payment method
- Print an order receipt
- Receive an order confirmation notification

---

## Features

- Display available menu items
- Add multiple items to an order
- Automatic total price calculation
- Cash or Card payment options
- Receipt generation
- Order confirmation notification
- Clean and modular code structure

## How It Works

1. Display the food menu.
2. User selects one or more menu items.
3. System stores selected items in an order.
4. User chooses a payment method.
5. Payment is processed.
6. Receipt is printed.
7. Confirmation message is displayed.
